# timeslottr Demo

This is a [Next.js](https://nextjs.org/) project that serves as a documentation and playground site for the `timeslottr` library.

## Local Development

1. Navigate to the demo directory:
   ```bash
   cd demo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment on Vercel

This demo is ready to be deployed on Vercel.

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and add a new project.
3. Import your repository.
4. **Important**: In the "Root Directory" settings, edit it to be `demo` (since this app lives in a subdirectory).
5. The build command (`next build`) and output directory (`.next`) should be detected automatically.
6. Click "Deploy".
